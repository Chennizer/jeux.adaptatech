Suggestion de jeu : Flow Lines

Utilise le modèle des sections Jeux & Gaming, avec un effet déclenché par la barre d’espace/switch.

Flow Lines (Présence → Direction)

Ce que c’est

Un champ plein écran de longues lignes continues qui ondulent comme de l’herbe dans le vent.

Interaction

  • Contrôle oculaire : courbe localement le flot (les lignes se plient vers le point regardé).
  • Switch / barre d’espace : chaque appui renforce le vent ; au relâchement, la force décroît progressivement pour revenir au calme.

Différences par rapport aux effets existants

  • Aucune particule
  • Pas de peinture
  • Aucun objet discret
  • Mouvement cohérent et global, sans explosions locales

Pourquoi c’est adapté pour PMLD/PIMD

  • Effet extrêmement régulateur
  • Lien clair cause → effet sans demander de précision
  • Permet une longue période d’engagement

Techniques p5.js suggérées

  • Champs de vecteurs
  • Bandes de lignes avec décalage basé sur le bruit
  • Variable de force globale avec décroissance
