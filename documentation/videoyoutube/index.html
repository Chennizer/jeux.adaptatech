Nouvelles fonctions — Sélection de vidéos YouTube

Les mêmes options générales que dans les autres activités sont disponibles (modes, image/son indicateur, etc.). Pour leur description détaillée, voir cette page .

Ajouter une vidéo par URL

Il est possible d’ajouter une vidéo YouTube en collant son URL dans le champ prévu, puis en cliquant sur « Ajouter URL ». La vignette et le titre apparaissent dans la liste.

Importer une playlist YouTube

Une playlist complète peut être importée via son URL. Les vidéos compatibles sont ajoutées automatiquement à la sélection avec miniatures et titres. L’état d’importation s’affiche sous le champ.

Mémoire locale

La sélection (y compris l’ordre) est mémorisée localement dans le navigateur. Au prochain chargement de la page (même appareil, même navigateur), la liste réapparaît automatiquement.

Réordonner les vidéos

L’ordre de lecture peut être ajusté avec les contrôles de réorganisation (par exemple, « haut / bas »). L’ordre affiché correspond à la lecture réelle.

Compatibilité et limites (YouTube)

Certaines vidéos ne peuvent pas être importées ni lues pour des raisons de droits ou de restrictions d’intégration (ex. contenus VEVO, vidéos bloquées par l’éditeur, limitations régionales). La majorité du contenu est utilisable et ce qui ne l'est pas est plutôt variable, il faut donc faire le test afin de savoir si cela fonctionne. Ces éléments sont ignorés automatiquement lors de l’ajout d’URL ou l’import d’une playlist.
