Guide de démarrage rapide

  1. La clé Ably de test est déjà préremplie dans le bloc config (champ ABLY_API_KEY).
  2. Choisissez une couleur/effet (vous pouvez tester localement avant de démarrer).
  3. Cliquez sur Commencer pour passer à l'écran épuré, puis utilisez la barre d'espace dans chaque onglet pour diffuser.
