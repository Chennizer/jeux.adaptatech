Guide de téléchargement

À propos

Ce guide explique comment récupérer les Gemini Storybooks pour les ajouter au site ou les partager en mode hors ligne. Les images illustrant chaque étape pourront être insérées dans les sections ci-dessous.

1. Ouvrir la page Gemini Storybooks

Depuis le site, cliquez sur la tuile Gemini Storybooks (ou le lien direct) pour afficher la bibliothèque. Vérifiez que vous êtes connecté si le contenu est réservé aux membres.

2. Choisir le format de téléchargement

*Note: depuis novembre 2025, il semble que google ait changé la façon de télécharger les fichiers. Ils sont maintenant directement téléchargé en haute définition via PDF. Cela dépendrait des utilisateurs, probablement que plusieurs versions sont en essai. Donc, si vous appuyez sur télécharger et que le fichier est directement téléchargé, vous pouvez passé à l'étape 4. Sur la plateforme de Gemini Storybooks, il est possible de faire une lecture avec génération vocale directe. Pour avoir accès à la version sur ce site permettant l'accès par switch ou contrôle oculaire, il faut télécharger les histoires. La première étape est de cliquer sur le bouton Imprimer.

Utilisez le bouton Télécharger pour ouvrir les options. Sélectionnez le paquet HTML offline pour une intégration directe au site ou un fichier PDF si vous souhaitez une version imprimable.

3. Lancer le téléchargement

Confirmez le format choisi. Un fichier compressé (.zip) est généré et téléchargé. Selon la taille de la collection, l’opération peut prendre quelques minutes.

4. Vérifier le contenu

Ouvrez le fichier .zip et vérifiez que chaque histoire contient les images, les fichiers audio (s’il y en a) et la page HTML. Corrigez les noms de fichiers si nécessaire pour éviter les espaces ou caractères spéciaux.

5. Ajouter au site

Une fois que vous avez le fichier PDF, vous pouvez vous rendre sur la page de votre choix, soit la page adaptée pour switch, soit la page adaptée pour contrôle oculaire. Vous pourrez ajouter votre PDF directement sur ces pages et choisir différentes options pour la lecture, dépendemment du type d'activité souhaitée et des capacités du lecteur.

6. Autres informations

Voici quelques informations supplémentaires concernant l'utilisation des PDF exportés :

  • Puisqu'il faut lire le texte directement dans l'image, il peut y avoir des erreurs dans la lecture (certaines lettres se ressemblent, etc.). De manière générale, c'est plutôt fidèle, et beaucoup plus performant en anglais.
  • La lecture du titre sur la première page est désactivée par défaut parce que c'est l'endroit où il y a le plus d'erreurs de reconnaissance.
  • La voix de synthèse vocale est contrôlée par votre ordinateur et non par le site web. Les voix disponibles varient selon l'appareil, et certaines peuvent être monotones. S'il y a de la demande, une voix générée par IA pourrait être intégrée.
