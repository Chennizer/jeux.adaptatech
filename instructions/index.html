Instructions

Panneau de contrôle pour les jeux avec switch

Voici les explications pour les options concernant les jeux de switch. À noter que le jeu est fonctionnel par défaut, il n'est pas nécessaire d'apporter des modifications aux options.

1. Choisir le mode de jeu

Le panneau de contrôle permet de choisir comment l'utilisateur interagit avec le jeu. Sélectionnez le mode le plus adapté en fonction des capacités et préférences de l’utilisateur :

  1. Un appui par vidéo : L'utilisateur doit appuyer sur sa switch pour avancer à la vidéo suivante ou à l'activité suivante.
  2. Un appui pour toute la liste : Avec un seul appui, toutes les vidéos ou activités se succèdent automatiquement.
  3. Mode intervalle : Le jeu s'arrête à intervalles réguliers, et l’utilisateur doit appuyer pour continuer. Vous pouvez définir le temps entre chaque pause dans la section temps d’intervalle en secondes.

2. Sélectionner des vidéos ou activités

Chaque jeu peut avoir plusieurs vidéos ou activités, et vous pouvez choisir lesquelles seront incluses dans la session :

  1. Appuyez sur le bouton Sélection de vidéos pour afficher les options disponibles.
  2. Sélectionnez les vidéos ou activités souhaitées pour l’utilisateur. Les vidéos sélectionnées seront mises en surbrillance. Par défaut, toutes les options sont activées
  3. Une fois vos choix faits, appuyez sur OK pour confirmer.

3. Choisir une image ou un son indicateur

Il peut être pertinent, dépendamment de l'utilisateur, d'avoir un indicateur visuel ou sonore pour lui signaler quand appuyer sur sa switch. Vous pouvez personnaliser cet indicateur :

  1. Cliquez sur Image indicatrice pour choisir une image adaptée ou entrer un texte personnalisé qui s'affichera à l'écran.
  2. Vous pouvez également choisir un son indicateur (par exemple, un gong ou une cloche) dans le menu Son. Si aucun son ne convient, il est possible d'enregistrer un son personnalisé grâce au bouton Enregistrer un son indicateur.

4. Choisir le mode d'affichage de l'image

Pour améliorer l’expérience visuelle de l’utilisateur, vous pouvez choisir un mode d’affichage particulier parmi les options suivantes :

  • Image normale : Affiche l’image sans aucun filtre ou effet.
  • Saturation verte : Accentue les tons verts, ce qui peut aider les utilisateurs ayant des préférences pour des contrastes verts.
  • Saturation rouge : Augmente les nuances de rouge pour renforcer les couleurs chaudes.
  • Saturation bleue : Met en avant les tons bleus, ce qui peut être utile pour créer une ambiance plus apaisante.
  • Contraste élevé : Renforce les contrastes entre les couleurs claires et foncées pour une meilleure lisibilité.
  • Noir et blanc : Convertit l’image en nuances de gris, idéal pour les utilisateurs sensibles aux couleurs.
  • Couleurs inversées : Inverse les couleurs pour créer un effet visuel unique et captivant.
  • Haute clarté : Augmente la luminosité générale de l’image, utile dans des environnements plus sombres.
  • Haute saturation : Accentue toutes les couleurs, rendant l’image plus vive et dynamique.

5. Démarrer le Jeu

Une fois toutes les options configurées :

  1. Appuyez sur Commencer pour démarrer la session de jeu.
  2. L’utilisateur interagira avec le jeu selon les paramètres que vous avez choisis (mode de jeu, vidéos, images, sons, etc.).

Conseils pour une utilisation optimale

  • Activez le mode plein écran du navigateur pour maximiser l’immersion du jeu (appuyez sur F11 ou FN + F11).
  • Si vous souhaitez recommencer ou revenir au menu principal, appuyez sur F5 pour actualiser la page.
  • Vous pouvez tester les diverses options pour déterminer ce qui vous convient, rafraîchir la page restaure la configuration par défaut.
